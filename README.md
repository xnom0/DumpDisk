**DumpDisk**

DumpDisk permet d'effectuer une copie conforme d'un support de stockage avec une vérification de la somme de contrôle (checksum) du support de stockage et du dump afin de vérifier l'intégrité du dump.

Le point important de l'outil est qu'il peut cloner un support de stockage sans écrire sur le disque source, il calcule également la vitesse de lecture / écriture (dans le dump) et le pourcentage de copie.

L'outil est compatible Windows et GNU/Linux.
